## 一、方略要点（五条，执行时不跑偏）

| # | 要点 | 含义 | 违反时的典型后果 |
|---|------|------|------------------|
| 1 | **合规与鉴权先于炫技** | 域名 → HTTPS →（视机房）备案 → 微信合法域名 → `wx.login` + Bearer；生产 `COSII_TRUST_DEV_HEADER=0` | 真机全挂或“能演示但不能上线” |
| 2 | **契约与状态真源优先** | API envelope、`DESIGN.md` Day3–8 状态语义优先于像素级设计稿 | 粉品牌色盖住 warning/error；支付与争议误导用户 |
| 3 | **先闭环再美化** | 无断链、无假成功、错误可重试；再谈对标稿瀑布流/凸起「+」 | 演示好看但信任崩塌 |
| 4 | **双轨并行：运维轨 + 产品轨** | 运维轨串行解锁微信；产品轨按 Office Hours 清单迭代 UI（不阻塞合规） | 要么永远只修 UI，要么只部署不验收 |
| 5 | **门禁不降级** | 合并前 `npm run verify:gates`（或至少 `test` + `smoke:miniprogram`） | 回归进主链 |

**首期目标（与评审前提门一致）：** 可信闲置成交 + 约妆骨架 + 订单/消息可追踪；**不以** 设计稿 100% 还原为上线定义。

| 0.5 | 生产环境变量 | `NODE_ENV=production`、`WECHAT_*`、`COSII_SESSION_SECRET`、`COSII_TRUST_DEV_HEADER=0` 等 | **小白：** [同上 §0.5](../tutorials/轨道0-合规步骤02-07小白教程.md) + [生产环境变量与真机联调（小白版）.md](../tutorials/生产环境变量与真机联调（小白版）.md)；**模板：** [`deploy/env.production.example`](../../deploy/env.production.example)、[`deploy/cosii-api.service.example`](../../deploy/cosii-api.service.example)；验收：`docs/go-live-checklist.md` |
| 0.6 | 真机一条主链 | `wx.login` → `POST /v1/auth/wechat-login` → 带 Bearer 调受保护接口 | **小白：** [轨道0-合规步骤02-07 §0.6](../tutorials/轨道0-合规步骤02-07小白教程.md) + [生产环境变量与真机联调（小白版）.md](../tutorials/生产环境变量与真机联调（小白版）.md)；验收：录屏或日志 + `trace_id`（脱敏） |
| 0.7 | 支付与回调（若首期含支付） | 回调 URL、验签、幂等、失败重试与告警 | **小白：** [轨道0-合规步骤02-07 §0.7](../tutorials/轨道0-合规步骤02-07小白教程.md)；工程验签与测试见 [P0教程 P0-4](./P0上线阻塞项实操教程（腾讯云服务器+域名）.md)；验收：沙箱/小额实付 + webhook 留样 |

**里程碑 M0：** 真机可在 **合法域名 + HTTPS** 下完成登录与至少一笔核心读接口（或支付链），且 **不依赖** `X-User-Id`。
